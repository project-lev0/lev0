Electron 기반 차세대 H-viewer

## 책임 한계
해당 프로그램은 직접적으로 컨텐츠를 제공하는것이 아닌 특정 사이트를 파싱하여 컨텐츠를 제공합니다.

사이트 편의성 개선을 목적으로 대가없이 무작위 사용자들에게 프로그램을 제공하고 있으며, 컨텐츠 제공 중단 및 컨텐츠 위법관련 문의는 컨텐츠를 제공하는 사이트들에 문의해주시기 바랍니다.

## 기능
- 로그인 지원
- 더 나은 디자인 지원
